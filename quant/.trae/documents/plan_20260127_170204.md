# Quant 爬虫 Web 前端界面完整方案

## 项目结构

```
quant/
├── web/                           # 前端界面目录
│   ├── index.html                 # 主页面（日期选择界面）
│   ├── styles.css                 # 样式表（响应式设计）
│   └── app.js                     # 前端逻辑
├── server.py                      # 轻量级后端服务
├── main.bat                       # 智能启动脚本
└── main.py                        # 现有爬虫逻辑（保持不变）
```

## 1. 前端界面设计 (web/)

### index.html 功能
- 单日期选择（HTML5 date input）
- 日期范围选择（开始/结束日期）
- 模式切换 Tab
- 提交按钮
- 状态提示区域

### styles.css 设计
- 现代化简洁 UI
- 响应式设计（移动端/桌面端）
- 清晰的视觉反馈

### app.js 功能
- Tab 切换逻辑
- 日期格式验证
- 调用后端 API 执行命令
- 显示执行状态

## 2. 后端服务 (server.py)

- Flask 轻量级 Web 框架
- API 端点：
  - `POST /api/run` - 执行爬虫命令
  - `GET /api/status` - 查询执行状态
- 调用 main_launcher.py

## 3. main.bat 启动脚本功能

- 启动 Flask 服务
- 检测服务是否成功启动
- 自动打开浏览器
- 彩色错误提示
- 支持开发模式

## 4. 使用流程

1. 用户运行 `main.bat`
2. 脚本启动 Flask 服务
3. 自动打开浏览器显示界面
4. 用户选择日期并提交
5. 后端调用 main.bat 执行爬虫
6. 实时显示执行状态
