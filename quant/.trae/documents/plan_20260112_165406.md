# 修改8.2主线题材输出格式（多行版）

## 目标
按照用户指定的多行格式输出8.2主线题材的股票信息

## 输出格式（4行）

```
第1行：{name}({code}) {continue_num}板/{limit_up_type} [{涨停时间}封/{open_num}开]
第2行：¥{最新价}({涨幅}) 成交{成交金额}/换手{换手率}
第3行：封单{封单额}/封成{封成比}/封流{封流比}
第4行：实流{实际流通市值} 题材:{概念标签}
```

## 示例

```
锋龙股份(002931) 12板/一字板 [09:25:00封/0开]
¥56.17(+10.01%) 成交2231.76万/换手0.20%
封单13.22亿/封成5924.4%/封流23.3%
实流56.87亿 题材:优必选收购+机器人概念+园林机械
```

## 字段计算规则

| 字段 | JSON | 格式化 |
|------|------|--------|
| 名称 | name | 直接 |
| 代码 | code | 直接 |
| 连板 | continue_num | +"板" |
| 涨停类型 | limit_up_type | 直接 |
| 涨停时间 | last_limit_up_time | Unix→HH:MM:SS |
| 打开次数 | open_num | null→0 |
| 最新价 | latest | ¥+.2f |
| 涨幅 | change_rate | ±.2f% |
| 成交金额 | trading_amount | /10000+.2f+"万" |
| 换手率 | turnover_rate | .2f+"%" |
| 封单额 | order_amount | /1e8+.2f+"亿" |
| 封成比 | order_amount/trading_amount×100 | .2f+"%" |
| 封流比 | order_amount/actual_currency_value×100 | .1f+"%"（1位小数） |
| 实际流通市值 | actual_currency_value | /1e8+.2f+"亿" |
| 概念标签 | reason_type | 直接 |

## 关键计算

- 封成比 = order_amount / trading_amount × 100
- 封流比 = order_amount / actual_currency_value × 100
