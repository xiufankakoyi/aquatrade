# Sandbox

Scratch space for investigations, reproductions, experiments, and temporary reports.

Code here is not part of the stable application surface. When a script becomes reusable, move it into `scripts/`, `core/`, `server/`, or `data_svc/` as appropriate.

- `root-scripts/`: one-off scripts moved out of the repository root.
- `tests/`: sandbox-only test experiments. Maintained tests belong in `test/`.
