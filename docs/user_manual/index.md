# User Manual

## Introduction
Aquatrade is an advanced backtesting and trading platform.

## Key Features
- **Backtesting**: Run strategies against historical data.
- **Optimization**: Use Genetic Algorithms to find optimal parameters.
- **Real-time Monitoring**: Watch strategy performance live.
- **Data Analysis**: Sentiment analysis, K-line charts, and more.
