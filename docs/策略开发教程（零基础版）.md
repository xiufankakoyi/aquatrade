### 什么是策略

策略就是一套买卖规则：
- **买入条件** - 什么时候买
- **卖出条件** - 什么时候卖
- **持仓条件** - 买多少、持有多久

### 双均线策略的规则

```
买入：当5日均线从下往上穿过10日均线时（金叉）
卖出：当5日均线从上往下穿过10日均线时（死叉）
```

```python
from core.strategies.strategy_layers import DualMAStrategy

# 双均线策略

# RSI策略
strategy = RSIStrategy(oversold=30, overbought=70)

# MACD策略
strategy = MACDStrategy()

# 布林带策略
strategy = BollingerStrategy()
```

### 写法2：声明式策略（推荐）

```python
from core.strategies.strategy_layers import SimpleStrategy
from core.strategies.utils.signal_utils import crossover, crossunder
import numpy as np

class MyStrategy(SimpleStrategy):
    """我的策略"""
    
    # 声明需要的因子
    required_factors = ['ma5', 'ma10', 'rsi_14']
    
    def _generate_signals(self, factors, trading_dates, stock_codes):
        # 获取因子
        ma5 = factors['ma5']
        ma10 = factors['ma10']
        rsi = factors['rsi_14']
        
        # 创建信号矩阵
        T, N = len(trading_dates), len(stock_codes)
        signals = np.zeros((T, N), dtype=np.int32)
        
        # 买入：金叉 且 RSI超卖
        golden = crossover(ma5, ma10)
        oversold = rsi[1:] < 30
        signals[1:][golden & oversold] = 1
        
        # 卖出：死叉
        death = crossunder(ma5, ma10)
        signals[1:][death] = 2
        
        return signals

# 创建策略
strategy = MyStrategy()
```

**信号值说明：**
- 0 = 观望
- 1 = 买入
- 2 = 卖出

**T+1执行：**
```python
signals[1:][condition] = 1  # 今天判断，明天执行
```

### 写法3：函数策略（灵活）

```python
from core.strategies.strategy_layers import FunctionStrategy, Position

def my_strategy(date, factors, position, history):
    """策略函数"""
    
    # 没有持仓时
    if not position.has_position:
        if factors['ma5'] > factors['ma10']:
            return 'buy', 0.1  # 买入10%仓位
    
    # 有持仓时
    else:
        if factors['ma5'] < factors['ma10']:
            return 'sell', None  # 全部卖出
    
    return 'hold', None

# 创建策略
strategy = FunctionStrategy(
    my_strategy,
    required_factors=['ma5', 'ma10']
)
```

## 信号工具

### crossover / crossunder - 金叉/死叉

```python
from core.strategies.utils.signal_utils import crossover, crossunder

# 金叉：快线从下往上穿过慢线
golden = crossover(ma5, ma10)
signals[1:][golden] = 1

# 死叉：快线从上往下穿过慢线
death = crossunder(ma5, ma10)
signals[1:][death] = 2
```

### above / below - 阈值检测

```python
from core.strategies.utils.signal_utils import above, below

# RSI > 70（超买）
overbought = above(rsi, 70)

# RSI < 30（超卖）
oversold = below(rsi, 30)

# 价格 > MA5
above_ma = above(close, ma5)
```

### rising / falling - 连续涨跌

```python
from core.strategies.utils.signal_utils import rising, falling

# 连续3天上涨
up_3d = rising(close, 3)

# 连续2天下跌
down_2d = falling(close, 2)
```

### in_range - 区间检测

```python
from core.strategies.utils.signal_utils import in_range

# RSI在30-70之间
normal_rsi = in_range(rsi, 30, 70)
```

### 示例4：市值筛选策略

```python
from core.strategies.strategy_layers import SimpleStrategy
import numpy as np

class MarketCapStrategy(SimpleStrategy):
    """市值筛选策略"""
    
    required_factors = ['close', 'total_mv', 'volume_ratio']
    
    def __init__(self, min_mv=200000, max_mv=600000, **kwargs):
        super().__init__()
        self.min_mv = min_mv
        self.max_mv = max_mv
    
    def _generate_signals(self, factors, trading_dates, stock_codes):
        close = factors['close']
        total_mv = factors['total_mv']
        volume_ratio = factors['volume_ratio']
        
        T, N = len(trading_dates), len(stock_codes)
        signals = np.zeros((T, N), dtype=np.int32)
        
        # 买入：市值20-60亿 且 量比>2
        buy_condition = (
            (total_mv >= self.min_mv) &
            (total_mv <= self.max_mv) &
            (volume_ratio > 2)
        )
        signals[1:][buy_condition[:-1]] = 1
        
        # 卖出：量比<1
        sell_condition = volume_ratio < 1
        signals[1:][sell_condition[:-1]] = 2
        
        return signals

strategy = MarketCapStrategy(min_mv=200000, max_mv=600000)
```

## 常见问题

### Q: 如何选择写法？

- **写法1**：完全不懂编程，只想快速验证
- **写法2**：想自定义规则，推荐使用
- **写法3**：需要复杂逻辑，比如持仓天数管理

### Q: 信号值0、1、2是什么意思？

- 0 = 观望
- 1 = 买入
- 2 = 卖出

### Q: 为什么用 `signals[1:]` 而不是 `signals[0:]`？

A股是T+1交易制度，今天发出的信号，明天才能执行。

### Q: 如何用AI帮我写策略？

把你的想法告诉AI，比如：
```
"我想写一个策略，当RSI小于30且量比大于2时买入，当RSI大于70时卖出"
```

AI会根据本教程生成正确的代码。

### Q: 如何优化参数？

```python
# 参数网格
param_grid = {
    'fast': [5, 10, 15],
    'slow': [20, 30, 60]
}

# 自动优化
results = engine.optimize(
    strategy_class=DualMAStrategy,
    param_grid=param_grid,
    start_date='2020-01-01',
    end_date='2024-01-01'
)

# 找到最优参数
best = max(results, key=lambda x: x.total_return)
print(f"最优参数: {best.params}")
```
